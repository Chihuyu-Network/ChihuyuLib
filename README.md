# chihuyu-snippets

Original utility library for chihuyu developement group.

## Usage
Add below code to `build.gradle.kts`.

```
repositories {
    maven("https://repo.hirosuke.me/snapshots/")
}
```

```
dependencies {
    compileOnly("love.chihuyu:chihuyu-utils:1.0.0-SNAPSHOT")
}
```
